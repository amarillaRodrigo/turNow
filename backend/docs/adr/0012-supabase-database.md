# 12. supabase-database

Date: 2026-01-13

## Status

Accepted

## Context

Necesitamos decidir la base de datos que utilizaremos para el backend

## Decision

Decidimos implementar Supabase(PostgreSQL) como base de datos principal. 
Nos da escalabilidad ademas de guardado en la nube y facil manejo de datos.

## Consequences

Las consultas a la base de datos seran mas rapidas y eficientes. Ademas de tener un sistema robusto y escalable para manejar grandes cantidades de datos.
Tambien todos los datos estaran seguros en la nube con backups automaticos.
